**Car Management System (Aston Team Project)**
A Java console application designed for managing, sorting, and validating custom car data.

**Tech Stack**
Java 17+
Maven (Dependency Management)
JUnit 5 & Mockito (Testing Frameworks)

**Testing Coverage**
The project includes a comprehensive test suite (JUnit 5):
Unit Tests: Isolated testing of validators and generators.
Integration Tests: Full round-trip testing (Generate -> Save -> Load -> Compare).
Negative Testing: Robust handling of empty files, invalid inputs, and boundary conditions.
Mocking: Use of Mockito to simulate user input and external dependencies.
